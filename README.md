# NumberToText
Делает число в виде текста.

## Requirements:
- Java

----------------------------------------------------------------------------------------------

## Usage:
#### Linux:
```
# Установка репозитория
$ git clone https://github.com/Ferius057/NumberToText.git

# Перейти в репозиторий
$ cd src

# Устоновка прав
$ chmod +x start.sh

$ Запуск
$ ./start.sh
```
#### Windows:
```
# Скоро...
```
